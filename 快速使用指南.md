# 🎮 德州扑克游戏 - 快速使用指南

## 🚀 启动应用

### 1. 启动服务器

```bash
cd "d:\GITHUB\Texas-hold'em\server"
npm run dev
```

✅ 服务器将在端口 **3001** 上运行

### 2. 启动客户端

```bash
cd "d:\GITHUB\Texas-hold'em\client"
npm run dev
```

✅ 客户端将在端口 **5173** 上运行

### 3. 访问游戏

- **本地访问**: <http://localhost:5173/>
- **局域网访问**: <http://192.168.112.118:5173/>

## 🎯 游戏流程

### 创建房间

1. 点击"**创建新游戏**"按钮
2. 设置游戏参数：
   - 游戏时长（分钟）
   - 最大玩家数（2-10人）
   - Straddle选项
   - All-in发牌次数
3. 点击"创建房间"
4. 自动进入游戏房间，获得6位房间ID

### 加入房间

1. 在首页输入**6位房间ID**
2. 按**回车键**或点击"加入游戏"
3. 自动进入房间
4. 选择座位（点击空座位）

### 开始游戏

1. 至少需要**2名玩家**入座
2. 主持人（房主）点击"**开始游戏**"按钮
3. 游戏自动发牌并开始

## ⌨️ 键盘快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **F** | 弃牌 | Fold - 放弃当前手牌 |
| **C** | 过牌/跟注 | Check/Call - 过牌或跟注 |
| **R** | 加注 | Raise - 打开加注面板 |
| **A** | All-in | 全押所有筹码 |
| **Esc** | 取消 | 关闭加注面板 |

💡 **提示**: 快捷键仅在轮到你操作时有效，且不会在输入框中触发

## 🎲 游戏操作

### 基本操作

#### 1️⃣ 弃牌（Fold）

- 点击"**弃**"按钮或按 **F** 键
- 放弃当前手牌，不参与本轮底池争夺
- ⚠️ 操作不可撤销

#### 2️⃣ 过牌（Check）

- 当前下注 = 当前最大下注时可用
- 点击"**过牌**"或按 **C** 键
- 不增加筹码，将操作权传给下一位

#### 3️⃣ 跟注（Call）

- 当需要跟注时显示
- 点击"**跟XXX**"或按 **C** 键
- 跟上当前最大下注

#### 4️⃣ 加注（Raise）

- 点击"**加**"按钮或按 **R** 键
- 打开自定义加注面板
- 可以使用：
  - **滑块**调整金额
  - **快捷按钮**（1/3池、1/2池、1x池、1.2x池）
  - **手动输入**金额
- 金额自动对齐到大盲的倍数

#### 5️⃣ All-in（梭）

- 点击"**梭**"按钮或按 **A** 键
- 全押所有剩余筹码
- 会创建边池（如果有多人）

## 🎨 界面元素说明

### 顶部信息栏

- **房间ID**: 6位代码，分享给朋友加入
- **在线玩家**: 当前房间人数
- **分享按钮**: 复制房间链接
- **退出按钮**: 离开房间

### 牌桌中央

- **公共牌**: 5张公共牌（翻牌、转牌、河牌）
- **底池**: 当前底池总额
- **边池**: All-in场景下的多个底池
- **游戏阶段**: 翻牌前/翻牌/转牌/河牌/摊牌

### 玩家位置

- **头像**: 玩家emoji图标
- **昵称**: 玩家名称（可简化显示）
- **筹码**: 剩余筹码数量
- **状态**: 行动中/弃牌/All-in/等待
- **手牌**: 仅你能看到自己的牌
- **位置标记**: BTN（庄家）、SB（小盲）、BB（大盲）

### 操作面板

- **倒计时**: 60秒操作时间
- **操作按钮**: 弃/过牌/跟/加/梭
- **加注滑块**: 自定义加注金额
- **快捷加注**: 一键加注到指定倍数

### 右侧边栏

- **游戏日志**: 最近20条操作记录
  - 🔴 红色 = 弃牌
  - 🟢 绿色 = 过牌
  - 🔵 蓝色 = 跟注
  - 🟡 黄色 = 加注
  - 🟣 紫色 = All-in
- **排行榜**: 玩家筹码排名

## 🎮 游戏规则

### 德州扑克基础

1. **手牌**: 每人2张底牌（仅自己可见）
2. **公共牌**: 5张公共牌（所有人可见）
   - 翻牌：3张
   - 转牌：1张
   - 河牌：1张
3. **目标**: 用2张手牌+5张公共牌组成最大的5张牌组合

### 牌型大小（从大到小）

1. **皇家同花顺** - A K Q J 10 同花色
2. **同花顺** - 连续5张同花色
3. **四条** - 4张相同点数
4. **葫芦** - 3张+1对
5. **同花** - 5张同花色
6. **顺子** - 连续5张
7. **三条** - 3张相同点数
8. **两对** - 2对牌
9. **一对** - 1对牌
10. **高牌** - 最大的单张

### 下注轮次

1. **翻牌前（Preflop）**: 发2张手牌后
2. **翻牌（Flop）**: 翻3张公共牌后
3. **转牌（Turn）**: 翻第4张公共牌后
4. **河牌（River）**: 翻第5张公共牌后
5. **摊牌（Showdown）**: 亮牌比大小

## ⚙️ 房间设置

### 游戏时长

- 范围：5-60分钟
- 到时自动结算
- 筹码最多者获胜

### 最大玩家数

- 范围：2-10人
- 超过人数可观战
- 观战者可在下一轮入座

### Straddle

- 盲注倍增机制
- BTN位可选择Straddle
- 增加游戏刺激性

### All-in发牌次数

- 控制All-in后发牌次数
- 1次 = 只发1次
- 2次 = 发2次取最好结果

## 🐛 常见问题

### Q: 看不到自己的手牌？

A: 请确保：

1. 你已经入座（不是观战）
2. 游戏已开始
3. 你未被淘汰

### Q: 按钮点不了？

A: 检查：

1. 是否轮到你操作（看倒计时）
2. 你是否已弃牌或All-in
3. 是否在"处理中..."状态

### Q: 断线后如何恢复？

A: 系统会自动尝试重连：

1. 等待自动重连（最多5次）
2. 或点击"手动重连"按钮
3. 重连后自动恢复游戏状态

### Q: 如何邀请朋友？

A: 两种方式：

1. 点击顶部"分享"按钮，复制链接
2. 告诉朋友6位房间ID

### Q: 筹码用完怎么办？

A: 可以申请补码（Rebuy）：

1. 点击自己的筹码显示
2. 选择补码金额
3. 等待下一轮开始

## 🎯 进阶技巧

### 1. 位置优势

- **BTN（庄家）**: 最后行动，信息最多
- **SB（小盲）**: 第一个行动，位置劣势
- **BB（大盲）**: 已投入盲注，保护底池

### 2. 加注策略

- **价值加注**: 手牌强时加注获取价值
- **诈唬加注**: 手牌弱时加注吓退对手
- **隔离加注**: 淘汰弱手，单挑强手

### 3. 底池管理

- **底池赔率**: 计算是否值得跟注
- **隐含赔率**: 考虑后续可能赢的筹码
- **底池控制**: 用下注控制底池大小

### 4. 读牌技巧

- **对手行动**: 快速跟注可能是听牌
- **下注大小**: 小注可能试探，大注可能强牌
- **游戏日志**: 查看对手历史操作习惯
